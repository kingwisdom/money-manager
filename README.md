
# Money Manager

A simple application for managing personal finances and tracking expenses.

## Features

- Track income and expenses
- Categorize transactions
- View spending summary
- Generate financial reports

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
git clone <repository-url>
cd money-manager
npm install
```

### Usage

```bash
npm start
```
